# BitBook

@author Thiago Milk Rodrigues

**Status do projeto:** Em desenvolvimento

## Tecnologias utilizadas
- Java
- Java Swing (JFrame)
- JDBC
- MySQL
- GitHub Desktop / GitHub

## Time de desenvolvedores
- Thiago Milk Rodrigues

## Objetivo do software
Sistema de controle de transações de criptomoedas, voltado para usuários que desejam registrar e acompanhar suas movimentações de compra e venda.

## Funcionalidades do sistema
- Cadastro de transações
- Listagem de transações em tabela
- Cálculo de ganhos e perdas (disponível no .txt gerado)
- Armazenamento de dados em banco de dados MySQL
- Interface gráfica com Java Swing
